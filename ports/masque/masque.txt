Introduction:

MASQUE.STX is a module for STinG, which allows any number of local machines
in a STinG LAN to access Internet simultaneously through a single connection
to an ISP.
------------------------------------------------------------------------------

This small text file is NOT the real 'Masque' document...!

That is supplied in two other files:

MASQUE.STG	;Ascii readable ST-Guide hypertext source
MASQUE.HYP	;Compiled ST-Guide hypertext

I strongly recommend using the compiled hypertext version, but the document
of Masque is fairly short and simple, so the STG source will suffice.

The actual module file for use with STinG is of course:

MASQUE.STX

Now read the docs and "Let the masquerade begin"...
